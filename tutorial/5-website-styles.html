Style body and html tags

Our body and html are not 100% high. We can check that using Developer Tools:

Note: height of the HTML and BODY on your screen might be different - it's ok.

We have to set height using CSS.


html,body {
    height: 100%
}

Check hegiht before and after in inspector.

As you can already see we can apply styles to more than one selectors at the same time (html and body)

Body background

Now when our body is 100% heigh, let's change our background from white color to image:


body{
    background-image:url(https://images.unsplash.com/photo-1550439062-609e1531270e?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=crop&w=1950&q=80);
}

Our image is too big now. Let's resize it:


background-size: cover;

And reposition:


background-position: center;

Since our image has different colors, it will be diffcult to type anything on it. Let's add some solid color layer:


background-color:#051f35;

And mix it with the image:


background-blend-mode: screen ;

Something new - asterix (*)

As you noticed our text disappeard. It actaully didn't disappeard but become invisible or rather difficult to read beacuse we have black text on dark background.

You already know that we can apply CSS to certain elements but what if we want to apply it to all elements one the website? Then we can use * selector which literally "everything". It will apply styles to each and every element on the webpage.

I.e. instead of changing color of our heading nad paragraphs indivudally, we can change collor of all elements in the website at once.


*{
    color: white;
}

Important dot (.)

You may noticed that before we were using class="" atribute in HTML tags and .classname in css fiels to apply styles only to element having this class only. In our previous steps we just used html and body tag name without class. This means that it will be applied to all tags. I.e. if we use p {color: blue} we will change color of all paragraphs on the website to blue.

.heading class

Lets move to .heading div. We want our div to be centered both vertically and horizaontally

Let's define it's size:


.heading{
    width: 600px;
    height: 400px;
}

And let's center it both vertically, as well as horizontally:


margin: auto;
position:absolute;
top: 0px; bottom: 0px; right: 0px; left: 0px;

Note: Don't worry if you don't understand some of the syntax. We will cover that soon. For now just remember that this is one of the way to postion div to the center.

Styling title

Let's center our title:


.title{
    text-align: center;
}

Increase the font size to 80px:


font-size: 80px;

Note: px is on of the available units. Read more about other CSS units here.

And change font ouf our title:


font-family: Courier New;

Note: you can change not only font kind but also other properties. Read more about other CSS fonts here.

And change it's weight to normal:


font-weight: normal;

We are almost done with title. However you may notice that we have a huge gap in between our title and subtitle.

This is because web browser automatically adds default margins to heading elements. Let's check this in inspector tool.

Let's remove bottom margin using css:


margin-bottom: 0px;

Note: each ellement in HTML has padding, border and margin, it's so called box model. Read more about box model here.

Styling subtitle

Again change size, position and font:


.subtitle{
    text-align: center;
    font-size: 30px;
    font-family: 'Courier New', Courier, monospace;
}

Button

Finally let's adjust our button. things you already know: width, height, font family, font, weight:


.register{
    width:180px;
    height:50px;
    font-weight: bold;
    font-family: 'Courier New', Courier, monospace;
}

Let's use margin to center our button:


margin: 50px auto;
display: block;

This new syntax for margin is shorter version, instead of writing following:


margin-top: 25px;
margin-right: 50px;
margin-bottom: 75px;
margin-left: 100px;

We can use:


margin: 25px 50px 75px 100px;

Which in our case means that we are setting top margin to 50px and all other to auto (to be decided by browser).

We also added new property - display:block. For now you just have to know that there are block and inline type of elements in CSS. Block-level elements always starts on a new line, while inline elemtns does not start a new line. Siimple test:



Paragraph 1
 
Paragraph 2

Span 1Span 2

Paragraphs starts in a new line (since by deafult they block elements), while spans are displayed next to each other - they are inline elements.

Let's change background of our button:


background-color: rgba(0,0,0,0);

Instead of providing a color name, we used it's RGB value , the last number is opacity value. 0 makes color totally transparent.

Note:Read more about CSS colors here

Finally , lets change border of our button:


border: 3px solid white;

And give it some curve:


border-radius: 5px;

Note:Read more about borders here

Our website is ready. But let's make it even more fantastic and add some animation. We will add some animation to our button when someone hover mouse pointer over it.


.register:hover{
    background-color: white;
    color: #222222;
}

As you can see we are applying style to .register class but also specyfing that we ant to apply style only when someone hover over this element.

It works cool, but how about make it more smooth?

Let's update both .register and .register:hover clasess accordingly:


.register{
    width:180px;
    height:50px;
    font-weight: bold;
    font-family: 'Courier New', Courier, monospace;
    margin: 50px auto;
    display: block;
    background-color: rgba(0,0,0,0);
    border: 3px solid white;
    border-radius: 5px;

    transition: background-color 1000ms, color 1000ms;
}

.register:hover{
    background-color: white;
    color: #222222;

    transition: background-color 1000ms, color 1000ms;
}
